# Cyber Event Report — demo

A self-contained, single-page HTML **event report** for the April 2026
ShinyHunters → Anodot → Snowflake supply-chain breach. It presents a multi-source
knowledge-graph view of the incident: a typed entity/relationship graph, source
**reliability tiering** (load-bearing vs reach-only), reconciled conflicting claims,
related and ongoing events, and adjudicated **false alarms** — incidents that
surfaced as candidates but were investigated and ruled out.

Open `index.html` in any browser, or view the published page via GitHub Pages.
The diagrams are generated with [ConceptViz](https://conceptviz.app/).
